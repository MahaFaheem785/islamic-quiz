# 🕌 Islamic Quiz Application

An interactive **Islamic Quiz Application** designed to test and improve knowledge about Islam in a simple and engaging way.

---

## 📌 Project Overview

This project is a quiz-based application where users can answer multiple-choice questions related to Islamic knowledge.

Islamic quiz apps are commonly used to help users learn topics like Quran, Hadith, history, and basic beliefs in an engaging way ([itch.io][1]).

---

## 🚀 Features

* ❓ Multiple-choice quiz questions
* 📚 Questions related to Islamic knowledge
* 🎯 Instant feedback on answers
* 🧠 Helps improve learning in a fun way
* 💻 Simple and user-friendly interface

---

## 🛠️ Technologies Used

* HTML / CSS
* React Js 
* Php

---

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/MahaFaheem785/islamic-quiz.git
```

2. Navigate to the project folder:

```bash
cd islamic-quiz
```

3. Run the project:




* If Web:
  Just open `index.html` in your browser

---

## 💡 How It Works

* User starts the quiz
* Questions are displayed one by one
* User selects an answer
* System checks correctness
* Score is calculated at the end
